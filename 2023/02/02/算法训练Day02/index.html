Leetcode977 有序数组的平方

1、题目链接：https://leetcode.cn/problems/squares-of-a-sorted-array/

2、题目描述：给你一个按非递减顺序 排序的整数数组 nums，返回每个数字的平方组成的新数组，要求也按非递减顺序排序。示例 1： 输入：nums = [-4,-1,0,3,10] 输出：[0,1,9,16,100] 解释：平方后，数组变为 [16,1,0,9,100]，排序后，数组变为 [0,1,9,16,100]；示例 2： 输入：nums = [-7,-3,2,3,11] 输出：[4,9,9,49,121]

3、标准答案：（2种）

  • 暴力排序：平方后快速排序，时间复杂度是 O(n + nlogn)。
  • 双指针法：数组是有序的，且数组平方的最大值就在数组的两端，不是最左边就是最右边，不可能是中间。此时可以考虑双指针法了，i指向起始位置，j指向终止位置。定义一个新数组result，和A数组一样的大小，让k指向result数组终止位置。如果A[i] * A[i] < A[j] * A[j]那么result[k--] = A[j] * A[j];如果A[i] * A[i] >= A[j] * A[j]那么result[k--] = A[i] * A[i]; 时间复杂度为O(n)（以空间换时间）。

4、总结：

  • 快速排序的写法：sort(nums.begin(), nums.end());

  • vector新数组的定义：vectorresult(length,0);

Leetcode209 长度最小的子数组

1、题目链接：https://leetcode.cn/problems/minimum-size-subarray-sum/

2、题目描述：给定一个含有 n 个正整数的数组和一个正整数 s ，找出该数组中满足其和 ≥ s 的长度最小的 连续 子数组，并返回其长度。如果不存在符合条件的子数组，返回 0。示例：输入：s = 7, nums = [2,3,1,2,4,3] 输出：2 解释：子数组 [4,3] 是该条件下的长度最小的子数组。

3、思路：

  • 暴力解法：循环遍历所有情况（两个for循环，不断的寻找符合条件的子序列，时间复杂度O(n^2)）

  • 滑动窗口法：不断的调节子序列的起始位置和终止位置，从而得出我们要想的结果（只用一个for循环，且这个循环的索引，一定是表示滑动窗口的终止位置）

    1.窗口内是什么？窗口就是满足其和 ≥ s 的长度最小的连续子数组。

    2.如何移动窗口的起始位置？如果当前窗口的值大于s了，窗口就要向前移动。

    3.如何移动窗口的结束位置？窗口的结束位置就是遍历数组的指针，也就是for循环里的索引。

    滑动窗口根据当前子序列和大小的情况，不断调节子序列的起始位置。从而将O(n^2)暴力解法降为O(n)。

4、总结：

  • 掌握滑动窗口法：不断的调节子序列的起始位置和终止位置，从而得出我们要想的结果。
  • 注意要根据题目意思，判断for循环中的循环索引，应该表示滑动窗口的起始位置，还是终止位置。

Leetcode59 螺旋矩阵II

1、题目链接：https://leetcode.cn/problems/spiral-matrix-ii/

2、题目描述：给你一个正整数 n ，生成一个包含 1 到 n2 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。示例:输入: 3 输出: [ [ 1, 2, 3 ], [ 8, 9, 4 ], [ 7, 6, 5 ] ]

3、思路：模拟整个螺旋的过程，由外向内一圈一圈的循环遍历：填充上行从左到右；填充右列从上到下；填充下行从右到左；填充左列从下到上。

4、总结：

  • （边界条件非常多，坚持左闭右开原则）坚持循环不变量原则；
  • 注意模拟转一圈的过程，四个for循环位置不能颠倒；此时的（i，j）即为实时位置，（startx，starty）即为起始位置；
  • 学习掌握二维数组的定义方法：vector<vector>res(n, vector(n,0));// 使用vector定义一个二维数组。

滑动窗口

  • 暴力解法时间复杂度：O(n^2) -> 滑动窗口时间复杂度：O(n)
  • 滑动窗口的精妙之处在于根据当前子序列和大小的情况，不断调节子序列的起始位置。从而将O(n^2)的暴力解法降为O(n)。

模拟行为

  • （边界条件非常多，坚持左闭右开原则）坚持循环不变量原则；
  • 对于边界调节超多，一波接着一波的判断，找边界，拆了东墙补西墙，好不容易运行通过了，代码写的十分冗余，毫无章法的题目，其实真正解决题目的代码都是简洁的，或者有原则性的。
